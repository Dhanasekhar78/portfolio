PULSE WIDTH MODULATION

Pulse width modulation (PWM) is one of the techniques through which one can transmit a message signal in the form of pulses. The pulses generated through this technique have different widths. These varying widths of pulses are used in switching electrical and electronic circuits. PWM technique is used in many applications like Motor speed control, LED brightness control to vary voltage given to it. Most oftenly variable resistors can be used. The main problem with this control is power dissipation across the variable resistor. This problem can be avoided by using Pulse Width Modulation (PWM). In this method the average voltage delivered to the load is controlled by varying the duty cycle of the rectangular wave, that is how the width of pulses are varied by controlling the signal applied to it. In this project 555 timer IC is operated in monostable mode for generating PWM signal.
