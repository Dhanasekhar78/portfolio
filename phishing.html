PHISHING WEBSITE DTECTION USING MACHINE LEARNING

Phishing attacks are a prevalent technique used by online scammers to obtain sensitive information from unsuspecting users. Detecting phishing websites is crucial to protecting users from these attacks. In this paper, we propose a machine learning approach for identifying phishing URLs. We compare the performance of several classification algorithms, including logistic regression, K-Nearest Neighbors, Support Vector Machine, Decision Tree, and Gradient Boosting Classifier. Our approach involves extracting features from the URLs and using them to train the models. We evaluate the performance of each algorithm using metrics such as accuracy, precision, recall, and F1-score. Our proposed method can be integrated into existing security systems to improve their ability to detect phishing attacks
